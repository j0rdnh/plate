# Plate

a CLI tool for creating and using templates!  
*This is a WIP :)*

### Usage

View/edit the config file with `$ plate config`  
├─ Home Directory  
└─ Editor  
Create new Plates with `$ plate etch`   
Use a Plate with `$ plate press`
