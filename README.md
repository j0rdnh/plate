# Plate

A CLI tool for creating and using templates!  
*This is a WIP :)*

## Usage

- View/edit the config file with `$ plate config`
  - home directory
  - editor
- Edit existing plates with `$ plate edit`
  - plate type
  - plate name
- Create new plates with `$ plate etch`
  - plate type
  - plate name
- Use a Plate with `$ plate press`
  - plate type
  - plate name
  - etching name
    - open etching
